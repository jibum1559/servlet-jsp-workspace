http 통신 코드

- http 서버 간의 통신에 사용되는 용도
- 클라이언트와 서버간의 요청과 응답을 교환하는 데 사용하는 코드
- 성공과 실패가 존재하며, 성공시 (200) 실패는 실패에 따라 각 다른 코드가 존재함 
	
	************면접 때도 많이 질문하는 부분이라 외우기 필수 *************
	>200 : 페이지 요청 성공!
	>404 : 문서를 찾지 못하는 경우 발생
		   url 주소 확인이나 캐시를 삭제한 후 다시 시작하기
		   url 주소 확인이란 내가 작성한 파일명과 파일명을 불러올 주소에 위치한 파일명이 불일치하지 않은지 확인할 것
	>405 : method를 수행하기 위해 해당 자원의 이용이 허용되지 않거나 메소드 명이 다를 경우 발생
	>500 : 서버 안에서 일어난 서버 내부 오류이므로 웹 서버가 요청을 수행할 수 없을 경우 발생
	>400 : 문법에서 오류가 있어서 잘못된 요청으로 판단하여 서버가 요청을 이해하지 못한 경우 발생
